# Financial Modeling & Valuation in Excel

This repository contains two Excel-based financial models built to practice and demonstrate financial statement analysis, valuation, forecasting, and Excel modeling skills.

## Projects Included

### Home Depot Financial Model
This model includes historical financial statement analysis, forecasting assumptions, annual and quarterly modeling, DCF valuation, WACC calculation, comparable company analysis, key metrics, scenario analysis, and supporting charts.

### Lowe's Financial Model
This model includes financial statement analysis, forecasting assumptions, valuation work, ratio analysis, and Excel-based financial modeling structure.

## Skills Demonstrated

- Three-statement financial modeling
- Financial statement analysis
- Forecasting and assumptions schedules
- DCF valuation
- WACC calculation
- Comparable company analysis
- Scenario analysis
- Ratio analysis
- Excel formatting and model organization
- Cross-sheet formula linking

## Purpose

These models were created as personal finance and valuation practice projects to strengthen Excel, financial analysis, and modeling skills. They are included as part of my portfolio for entry-level analyst roles in finance, banking, pensions, risk, and business analytics.
